# CMS前台网站UI设计Prompt

## 项目背景
当前项目是一个CMS内容管理系统，包含前台展示网站和后台管理系统。需要为前台网站设计现代化、美观的UI界面，参考主流资讯类和新闻类网站的设计风格。

## 参考网站分析
- **36氪**：现代简约的布局，突出内容，清晰的信息层次，响应式设计
- **数字尾巴**：卡片式设计，社区互动元素，精致的视觉效果
- **Smashing Magazine**：干净的排版，专业的技术内容展示，良好的阅读体验
- **CSS-Tricks**：技术博客风格，简洁明了，代码展示友好
- **UI设计网**：商业化风格，视觉冲击力强，注重用户引导

## 设计目标
创建一个现代化、专业的资讯类网站，具有良好的用户体验和视觉效果，符合当代Web设计趋势。

## 整体布局
1. **顶部导航栏**：
   - 固定在顶部，滚动时保持可见
   - 包含网站logo、主导航菜单、搜索框
   - 响应式设计，移动端显示汉堡菜单

2. **焦点图区域**：
   - 首页顶部展示，突出重要内容
   - 轮播效果，支持自动播放和手动控制
   - 图片+标题+摘要的组合展示

3. **主内容区**：
   - 三栏布局：左侧主内容，右侧边栏
   - 主内容区采用卡片式设计，展示文章列表
   - 按栏目分类展示内容

4. **侧边栏**：
   - 热门文章推荐
   - 标签云
   - 推荐阅读
   - 订阅/关注区域

5. **页脚**：
   - 网站信息
   - 导航链接
   - 版权信息
   - 社交媒体链接

## 颜色方案
- **主色调**：深蓝色 (#1a365d) - 代表专业、可信赖
- **辅助色**：橙色 (#ed8936) - 用于强调和交互元素
- **中性色**：
  - 白色 (#ffffff) - 背景
  - 浅灰 (#f7fafc) - 卡片背景
  - 中灰 (#e2e8f0) - 分割线
  - 深灰 (#4a5568) - 次要文本
  - 黑色 (#1a202c) - 主要文本

## 排版风格
- **标题**：使用无衬线字体，粗体，清晰的层次结构
- **正文**：使用易读的无衬线字体，适当的行高和字间距
- **字体大小**：
  - H1: 2.5rem (40px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - 正文: 1rem (16px)
  - 小文本: 0.875rem (14px)

## 组件设计
1. **文章卡片**：
   - 圆角设计 (8px)
   - 轻微阴影效果
   - 悬停时的微妙动画
   - 图片+标题+摘要+元信息的布局

2. **按钮**：
   - 主按钮：填充式，主色调
   - 次要按钮：边框式，中性色
   - 悬停效果：颜色变化和轻微放大

3. **导航菜单**：
   - 水平排列，间距均匀
   - 活跃状态有明确标识
   - 下拉菜单支持二级导航

4. **标签**：
   - 圆角设计
   - 轻量级背景色
   - 可点击，有悬停效果

5. **分页**：
   - 居中显示
   - 当前页有明确标识
   - 支持前后页导航

## 交互设计
- **悬停效果**：卡片、按钮、链接等元素有微妙的悬停动画
- **滚动效果**：平滑滚动，导航栏滚动时变化
- **加载动画**：内容加载时显示骨架屏或加载指示器
- **响应式设计**：适配不同屏幕尺寸

## 响应式设计
- **桌面端**：三栏布局，完整功能
- **平板端**：两栏布局，适当调整间距
- **移动端**：单栏布局，汉堡菜单，优化触摸体验

## 技术实现
- 使用Bootstrap 5作为基础框架
- 自定义CSS变量管理颜色和间距
- 使用Flexbox和Grid布局
- 适当使用JavaScript增强交互体验
- 确保性能优化，加载速度快

## 页面优化
- 图片懒加载
- 代码压缩
- 字体优化
- 响应式图片
- 减少HTTP请求

## 设计原则
- **一致性**：统一的设计语言和视觉风格
- **清晰性**：信息层次分明，易于理解
- **可用性**：良好的用户体验和交互设计
- **美观性**：现代、专业的视觉效果
- **性能**：快速加载，流畅交互
